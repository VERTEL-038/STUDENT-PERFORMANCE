### Генератор отчетов об успеваемости студентов

Консольное приложение для анализа успеваемости и посещаемости студентов

---

### Выполнили

Ахметов Динар 526989 - utils.py, main.py, data.py, ui.py

Закревский Иван 422237 - README.md, report.py

---

### Функционал

1. Вывод отчета об успеваемости студентов:
- Средний балл каждого студента
- Процент посещаемости каждого студента
- Студент с самым высоким и самым низким средним баллом
- Студeнт с самой высокой и самой низкой посещаемостью
- Наиболее и наименее посещаемое занятие
2. Просмотр тестовых данных
3. Выход из программы

---

### Структура проекта

1. main.py - точка входа в программу; запускает главное меню и обрабатывает выбор пользователя
2. ui.py - модуль пользовательского интерфейса: отображение меню и вывод тестовых данных
3. report.py - модуль генерации отчета: расчет и вывод статистики по студентам и занятиям
4. utils.py - вспомогательные функции: вычисление среднего арифметического оценок и расчет процента посещаемости
5. data.py - тестовые данные: список студентов (с оценками и посещаемостью) и список занятий

---

### Описание работы

Программа использует простые структуры данных Python для хранения информации о студентах и занятиях. Основная логика строится на:

- Переборе списков – для вычисления средних баллов, процентов посещаемости и поиска экстремумов
- Вспомогательных функциях – инкапсулируют расчеты, что делает код модульным и удобным для тестирования
- Генерации отчета – все вычисления выполняются за один проход по данным, после чего результаты выводятся в консоль в структурированном виде
